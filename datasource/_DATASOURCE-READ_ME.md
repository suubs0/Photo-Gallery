# Datasource Read Me

In modern web design, content (text for example) is not stored inside a web page. There is way too much text content to manage across the [billions of web pages](https://www.worldwidewebsize.com/) that are online to be able to have humans manage them all.

The content is stored inside special files called "databases" and often require computers dedicated to that single task: providing a structured way to house huge amounts of content easily accessible by other computers.

[See this LinkedIn Learning video for more info](https://www.linkedin.com/learning/php-with-mysql-essential-training-1-the-basics/mysql-introduction-14187622)
